# 📊 Superstore Sales Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive sales dashboard built using Power BI to analyze Superstore sales performance, profit trends, customer segments, and regional insights.


## 🛠 Tools Used

- Power BI
- Microsoft Excel
- Data Visualization
- Data Analysis

## 📊 Dashboard Insights

* Sales and profit analysis by region
* Category-wise and sub-category performance
* Customer segment analysis
* Monthly sales trends
* Interactive filters for better insights



## 📂 Dataset

The dataset used in this project is the Superstore Sales dataset containing information about orders, sales, profit, customers, regions, and product categories.
